# Changelog

All notable changes to this project are documented here. The format is based on
[Keep a Changelog](https://keepachangelog.com/en/1.1.0/), and this project adheres to
[Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0]

### Added

- Initial release: `jsesc` and `jsesc_with` — a zero-dependency, `no_std` port of the
  `jsesc` npm package's string escaping, with quotes, wrap, es6, escape_everything, minimal,
  is_script_context, json, and lowercase_hex options.
